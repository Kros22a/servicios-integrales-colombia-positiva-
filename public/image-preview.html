Previsualización de Imágenes

Instrucciones para Agregar Imágenes:

  1. Coloque sus imágenes en las carpetas correspondientes dentro de public/assets/images/
  2. Las rutas de las imágenes en su código deben ser relativas al directorio public, por ejemplo: /assets/images/services/construccion.png
  3. Para previsualizar una imagen, simplemente colóquela en la carpeta correcta y actualice esta página
  4. Cuando esté satisfecho con las imágenes, puede hacer commit y push a GitHub
